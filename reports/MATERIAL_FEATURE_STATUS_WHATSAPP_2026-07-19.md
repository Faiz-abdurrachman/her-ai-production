## 4. Status Fitur Pengaturan dan Ganti Password

### 4.1 Pengaturan Peserta 🟡

Halaman Pengaturan peserta **sudah tersedia secara visual**, tetapi belum berfungsi penuh.

**Bagian yang terlihat:**

- Profil Publik.
- Keamanan Akun.
- Preferensi Notifikasi.
- Tampilan dan Aksesibilitas.
- Upload dan hapus foto.
- Tombol Simpan Perubahan.

**Kekurangan:**

- Data masih berupa contoh hard-coded atas nama Aisyah Putri.
- Menu Keamanan, Notifikasi, dan Aksesibilitas belum aktif.
- Tombol Simpan, upload, dan hapus foto belum tersambung ke backend.
- Halaman Pengaturan masih berupa prototype visual.

**Catatan:**

- Edit profil dasar sudah tersedia di halaman profil terpisah.
- Data yang dapat diubah melalui halaman profil meliputi nama, email, WhatsApp, alamat, dan link CV.
- Admin Global Settings sudah tersedia, tetapi merupakan fitur yang berbeda dari Pengaturan peserta.

### 4.2 Ganti Password Peserta ❌

Fitur ganti password mandiri untuk peserta **belum tersedia**.

Kondisi saat ini:

- Peserta hanya dapat login menggunakan password yang dibuat atau diprovision oleh panitia.
- Belum ada form password lama, password baru, dan konfirmasi password.
- Fungsi `setParticipantPassword` di GAS hanya dirancang untuk pembuatan password pertama, bukan mengganti password yang sudah ada.
- Fungsi tersebut belum dibuka sebagai action GAS.
- Pembuatan password mandiri juga dinonaktifkan di frontend.
- Jika password peserta perlu diganti, saat ini harus dilakukan melalui reset atau provisioning ulang oleh admin/panitia.
